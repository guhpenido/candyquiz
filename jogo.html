◉ Títulos e subtítulos

Títulos e subtítulos são elementos extremamente comuns em HTML para vários sites. Esses elementos textuais nos trazem melhor navegação e a apresentação do site. As tags de título sempre têm o h seguido de um número que é o índice de "importância" do título, do h1(título) ao h6 (subtítulo).

◉ Imagens

Para inserir uma imagem no HTML basta utilizar a tag "img" com o atributo src. Ou seja: o atributo src, ou source, vai conter a url da imagem que será inserida.

Para inserir uma imagem local, podemos apenas incluir o nome da imagem com sua extensão.

A tag img apresenta o atributo alt, que é utilizado para especificar um texto alternativo que é renderizado quando a imagem não é renderizada, ou muitas vezes é utilizado por programas que leêm o conteúdo desse atributo para usuários cegos.

Os parágrafos apresentam quebra de linha automática.

◉ Hiperlinks

Os hiperlinks são links básicos criados envolvendo o texto que você quer transformar em um link dentro de um elemento. Ele é utilizado dando-lhe um atributo href, que conterá o endereço da Web para o qual você deseja que o link aponte.

◉ Divs - Containers de elementos

O elemento de divisão (div) é um container para conteúdos que de certa forma não representa nada. Ele geralmente é utilizado como forma de agrupamento de elementos para serem estilizados de forma conjunta.

◉ Classes e ID's

As classes e os ID's são formas de identificar grupos de elementos, para principalmente serem estilizados. Através delas, pode-se atribuir formatação e estilos a elementos.A única diferença entre elas é que as classes podem se atribuir estilos a vários elementos de uma vez, já os ID's, devem ser únicas para cada elemento.

Para inseri-los, você deve colocar class="nome-da-classe" (o mesmo para ID) dentro da abertura da tag.Para se referir às classes e ID's em códigos CSS, deve-se utilizar #nome-do-id para ID's, e .nome-da-classe para classes.
