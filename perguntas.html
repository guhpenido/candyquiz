Estrutura básica do HTML

Essa estrutura é praticamente imutável. Sempre será dessa forma e você sempre começará seu HTML dessa forma.

As tags demonstradas na foto podem ser definidas dessa forma:

  • Definição do documento (doctype)
  • Cabeçalho (head)
  • Corpo (body)

Títulos e subtítulos

Títulos e subtítulos são elementos extremamente comuns em HTML para vários sites. Esses elementos textuais nos trazem melhor navegação e a apresentação do site. As tags de título sempre têm o h seguido de um número que é o índice de "importância" do título, do h1(título) ao h6 (subtítulo).

HTML

Imagens

Para inserir uma imagem no HTML basta utilizar a tag com o atributo src. Ou seja: o atributo src, ou source, vai conter a url da imagem que será inserida.

Para inserir uma imagem local, podemos apenas incluir o nome da imagem com sua extensão.

A tag img apresenta o atributo alt, que é utilizado para especificar um texto alternativo que é renderizado quando a imagem não é renderizada, ou muitas vezes é utilizado por programas que leêm o conteúdo desse atributo para usuários cegos.
